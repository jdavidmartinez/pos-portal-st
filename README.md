# LANDING-PAGE

EmpleApp Landing-page

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

***

## Available Scripts ##

In the project directory, you can run:

Install dependencies:

```bash
npm install
```

Run server:

```bash
npm start
```

Run test:

```bash
npm run test
```

Run build:

```bash
npm run build
```
